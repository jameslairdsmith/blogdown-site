Customising CV/resume pagedown style

This will hopefully be helpful for those looking to make their pagedown CV look a little nicer, but don’t want to have to go through the trial and error involved in CSS changes.

Here is some CSS that should appear, but not run:

.blocks {
  break-inside: avoid;
}

This will get the bullet points to line up nicely:

[data-id="main"] li  {
  margin-left: 20px; 
  list-style-position: inside;
}
